# Journal de bord — Sessions Claude Code

Ce dossier contient une entrée par session significative du binôme PO/Claude Code.

## Convention de nommage

```
YYYY-MM-DD-<sujet-court>.md
```

Exemples :
- `2026-04-27-bootstrap-j0.md`
- `2026-05-03-skeleton-skill-j1.md`
- `2026-05-10-db-query-et-connexion.md`

## Quand créer une entrée

À la **fin de chaque session significative** — c'est-à-dire toute session où :
- Du code ou de la doc a été produit
- Des décisions ont été prises
- L'état du projet a changé

Les sessions purement informatives (questions ponctuelles, lecture de doc) ne nécessitent pas d'entrée.

## Format

Voir le format complet dans [`docs/state/CONTRIBUTING-CLAUDE-CODE.md §5`](../state/CONTRIBUTING-CLAUDE-CODE.md).

## Routine de lecture en début de session

Lire **uniquement la dernière entrée** pour reprendre le fil. Si la dernière entrée date de plus d'une semaine, lire aussi l'avant-dernière pour recapturer le contexte.
