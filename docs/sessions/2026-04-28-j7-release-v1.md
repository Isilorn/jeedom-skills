## Découvertes / surprises

- Le setup interactif `setup.py` est moins efficace que l'orchestration native de Claude Code : Claude détecte SSH + `~/.my.cnf` automatiquement sans passer par l'assistant. Le `setup.py` reste utile pour les utilisateurs qui n'ont pas Claude Code.
- Cas 03 (agendas bureau) : cause racine = condition incomplète dans le scénario "Invités" (éléments 71 et 65), absence de vérification de présence pour Alice et Bob. Diagnostic en ~11 min 41 s.
- Plugin Discord Link (cas 08) : daemon actif, 11 équipements, incident de janvier détecté dans les logs — ~43 s.
